Maskfy 3 - Mask Simple

A Javascript library without a dependencies... Compatibility with Vanilla, React, Vue, Angular and wherever. With only 1kb (gzip) code. It's also well accepted on mobile devices

Install / Usage

Add Maskfy file

<script type="module">
  import { maskfy } from './dist/maskfy.js'
  document.querySelector('#Plate').addEventListener('input', (e) => {
    const valueMask = maskfy(e.target.value, {mask: 'AAA 9A99'});
    e.target.value = valueMask.toUpperCase();
  });
</script>

or

  import { maskfy } from 'https://raw.githubusercontent.com/figuarnieri/maskfy/refs/heads/master/dist/maskfy.js'
